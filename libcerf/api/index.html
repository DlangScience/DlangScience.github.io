    API documentation

    Module Description
    libcerf.erfcx_ Compute erfcx(x) = exp(x^2) erfc(x) function, for real x, using a novel algorithm that is much faster than DERFC of SLATEC. This function is used in the computation of Faddeeva, Dawson, and other complex error functions.
    libcerf.err_fcts Computate Dawson, Voigt, and several error functions, based on erfcx, im_w_of_x, w_of_z as implemented in separate files.
    libcerf.im_w_of_x Compute complex error functions, based on a new implementation of Faddeeva's w_of_z. Also provide Dawson and Voigt functions.
    libcerf.testutils Internal test utilities. Use flags -unittest -debug=libcerf to receive debug info.
    libcerf.w_of_z Computation of Faddeeva's complex scaled error function, w(z) = exp(-z^2) * erfc(-i*z), nameless function (7.1.3) of Abramowitz&Stegun (1964), also known as the plasma dispersion function.
