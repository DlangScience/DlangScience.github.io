    Struct MeanSD

    Output range to compute mean, stdev, variance online. Getter methods for stdev, var cost a few floating point ops. Getter for mean costs a single branch to check for N == 0. Relatively expensive floating point ops, if you only need mean, try Mean. This struct uses O(1) space and does *NOT* store the individual elements.

    Properties

    Name Type Description
    mean [get] double
    mse [get] double Mean squared error. In other words, a biased estimate of variance.
    N [get] double
    stdev [get] double
    sum [get] double
    toMean [get] Mean Converts this struct to a Mean struct. Also called when an implicit conversion via alias this takes place.
    toMeanSD [get] MeanSD Simply returns this. Useful in generic programming contexts.

    Examples

    MeanSD summ;
summ.put(1);
summ.put(2);
summ.put(3);
summ.put(4);
summ.put(5);
assert(summ.mean == 3);
assert(summ.stdev == sqrt(2.5));
assert(summ.var == 2.5);
