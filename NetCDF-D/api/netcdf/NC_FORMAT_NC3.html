    Extended format specifier returned by nc_inq_format_extended() Added in version 4.3.1. This returns the true format of the underlying data. The function returns two values 1. a small integer indicating the underlying source type of the data. Note that this may differ from what the user sees from nc_inq_format() because this latter function returns what the user can expect to see thru the API. 2. A mode value indicating what mode flags are effectively set for this dataset. This usually will be a superset of the mode flags used as the argument to nc_open or nc_create.
